## 💻 Sobre o projeto

Projeto Web para validação de CEP, tendo sistema de login, pequeno sistema de cadastramento e grid.

---

## 🎨 Subir projeto 

Restaurar os pacotes Nugets.

Configurar migrations:

    $ Enable-Migrations

    $ Add-Migration subindo

    $ Udate-Database
